# Production_Inventory_Control_System
Sistem ini menggunakan struktur data berbasis FIFO untuk mengelola persediaan komponen. Setiap komponen memiliki histori stok masuk dan stok keluar. Proses produksi mengajukan permintaan komponen yang harus disetujui oleh warehouse sebelum stok dikeluarkan menggunakan metode FIFO.
